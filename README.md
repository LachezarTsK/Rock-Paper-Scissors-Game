Rock Paper Scissors is a two-player game - here played against the computer - where the players simultaneously choose from three possibilities: rock, paper, scissors. Rock wins over (crushes) scissors, scissors wins over (cuts) paper, and paper wins over (covers) rock. If both players have the same choice, it is a draw.

To run the application, click here: https://lachezartsk.github.io/Rock-Paper-Scissors-Game/
